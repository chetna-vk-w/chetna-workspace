# Patterns - Learned Heuristics

> *"Galti se seekho, par dobara mat karo."*

## Business Patterns

### Pattern: Speed to First Rupee
- **Context**: New venture, low capital
- **Pattern**: Validate demand before building
- **Trigger**: Below ₹50K stage
- **Result**: Faster feedback, less waste
- **Source**: Experience
- **Tags**: #pattern #category/business #stage/early

### Pattern: Recurring Revenue First
- **Context**: Sustainable income building
- **Pattern**: Prioritize subscriptions/retainers over one-time projects
- **Why**: Predictability compounds
- **Tags**: #pattern #category/business #priority/high

### Pattern: System Before Scale
- **Context**: Growth phase
- **Pattern**: Document and automate before hiring/expanding
- **Risk Avoided**: Chaos at scale
- **Tags**: #pattern #category/operations

## Technical Patterns

### Pattern: Read Before Ask
- **Context**: Problem solving
- **Pattern**: Check files, context, docs before asking human
- **Why**: Respects time, shows competence
- **Tags**: #pattern #category/technical #habit/good

### Pattern: Validate Before Build
- **Context**: Feature development
- **Pattern**: Test assumptions with smallest possible experiment
- **Tags**: #pattern #category/technical #risk/mitigation

### Pattern: Semantic Memory First
- **Context**: Information retrieval
- **Pattern**: Always search memory before external search
- **Why**: Contextual continuity, personal relevance
- **Tags**: #pattern #category/memory

## Communication Patterns

### Pattern: Match Energy
- **Context**: All interactions
- **Pattern**: Mirror communication style (Hinglish ↔ English, short ↔ detailed)
- **Why**: Natural flow, better connection
- **Tags**: #pattern #category/communication

### Pattern: Actions > Words
- **Context**: Demonstrating value
- **Pattern**: Show through doing, not promising
- **Tags**: #pattern #category/trust

### Pattern: Silent When Unneeded
- **Context**: Group chats
- **Pattern**: Don't respond to every message
- **Trigger**: Casual banter, already answered, just "yeah"
- **Tags**: #pattern #category/communication #habit/good

## Decision Patterns

### Pattern: Reversible = Fast
- **Context**: Decision making
- **Pattern**: If undoable, decide quickly
- **Tags**: #pattern #category/decisions

### Pattern: Pre-Mortem for Big
- **Context**: Irreversible decisions
- **Pattern**: Imagine it failed in 12 months, work backwards
- **Tags**: #pattern #category/decisions #risk/mitigation

## Health Patterns

### Pattern: Flag Before Pile
- **Context**: Human overworking
- **Pattern**: Notice stress signals, suggest break before adding tasks
- **Tags**: #pattern #category/health #relation/care

### Pattern: Small Consistent > Big Sporadic
- **Context**: Health habits
- **Pattern**: 20 min walk daily > 2 hour gym once a week
- **Tags**: #pattern #category/health
