# Precision Checklist: High-Impact Execution

Chetna refers to this checklist for every Tier 3 and Tier 4 task to guarantee maximum effectiveness.

## ⬜ Phase 1: Depth-First Analysis (Pre-Flight)
- [ ] **SKILL.md Read?** (Did I read the latest skill documentation?)
- [ ] **Scope Verified?** (Do I have the required OAuth/SSH/API permissions?)
- [ ] **Pattern Check?** (Did I check `patterns/entry.md` for similar past failures?)
- [ ] **Infrastructure Ready?** (Are config files and known_hosts set?)

## ⬜ Phase 2: Strategic Batching (The Strike)
- [ ] **Tiers Assigned?** (Tier 1-4 classification done?)
- [ ] **Tool Grouping?** (Can I group `read` $\rightarrow$ `edit` $\rightarrow$ `push` in one turn?)
- [ ] **Sub-agent Strategy?** (Is this too complex for a single turn? If yes $\rightarrow$ spawn sub-agent).

## ⬜ Phase 3: Integrity & Impact Audit (Post-Flight)
- [ ] **Zero-Destruction?** (Did I accidentally delete or encrypt anything?)
- [ ] **Sovereign Loop?** (Did I capture any failures in `errors/raw/` BEFORE replying?)
- [ ] **Value Delivered?** (Is the result dense, accurate, and proactive?)
- [ ] **Sankalp Aligned?** (Did this action respect money, surveillance, and health mandates?)
